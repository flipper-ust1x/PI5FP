BOOTLOADER
PINN is the chosen bootloader for this firmware. The reason PINN was chosen was for the recovery options and the dual boot options that allows us to put the full os,
and the recovery/fallback os(safe mode/if ubuntu fails) into the same sd card. for this firmware PINN has been modified by changing the background and icons.

Things added/changed with PINN
  >changed background to raspberry pi logo
  >changed menu color to black text on white
  >changed prompts
____________________________________________
OS
Ubuntu is the chosen OS for this firmware, this version of Ubuntu is optimized for the pi 5. adding utilities for the pi (through terminal) and adding interfaces
that let it run better. The Ubuntu theme has been changed heavily, the changes have been listed below.

Things added/changed on Ubuntu
  >taskbar is now on the bottom of the screen
